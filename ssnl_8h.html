Detailed Description

The functions of this library encapsulated the server side system programming level details of sending and receiving data over TCP sockets.

◆ tcp_client_error()

void tcp_client_error ( tcp_client *  client,
char **  ptr 
)
Parameters
[in]clientaddress of tcp_client type structure
[out]ptraddress of char* type pointer where the base address of the error string is to be kept
Precondition
before calling tcp_client_error, you must first call tcp_client_failed to determine if an operation has failed and an error mesasge has been generated. If an operation didn't failed, no error would be generated.
Postcondition
if error message exists, base address of the string with error message will be assigned to the pointer to which ptr is pointing to
if error message does not exist or enough memory to store message is not available, NULL will be assigned to the pointer to which ptr is pointing to
Note
when the error message is no more required, the memory should be released using the free function. The argument to the free function should be the base address of the string that contains the error message
Examples
app-server.c.
